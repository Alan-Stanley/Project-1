# 🛒 E-commerce Website

A static e-commerce website built using HTML, CSS, and JavaScript. This project demonstrates a modern user interface with product listings, categories, and customer testimonials, simulating a real-world online shopping experience.

---

## 🚀 Features

- Product grid layout for displaying items
- Featured and latest products sections
- Customer testimonial section
- Navigation bar and footer design
- Responsive layout for desktop and mobile screens
- Clean and structured UI design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (basic)
